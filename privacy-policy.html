Política de Privacidade

A sua privacidade é uma prioridade para nós. Esta política descreve como o aplicativo Bíblia Offline de Estudo gerencia informações e protege sua experiência de usuário.

1. Coleta e Uso de Dados

O aplicativo foi projetado para funcionar prioritariamente de forma offline. Não coletamos dados de identificação pessoal (como nome, telefone ou endereço) para o funcionamento das ferramentas de leitura e estudo.

  • Banco de Dados Local: Suas marcações, notas e históricos de busca (incluindo termos em Hebraico/Português) são armazenados localmente no seu dispositivo via SQLite e não são enviados para nossos servidores.
  • Logs de Erro: Podemos utilizar ferramentas técnicas anônimas para monitorar travamentos e melhorar a estabilidade do sistema.

2. Publicidade (Google AdMob)

No momento, o aplicativo não exibe publicidade. Contudo, informamos que planejamos implementar o Google AdMob no futuro para viabilizar o desenvolvimento e manutenção do projeto.

Quando ativos, os anúncios poderão utilizar identificadores exclusivos do seu dispositivo para personalizar o conteúdo publicitário. O processamento desses dados será regido pelas políticas do Google AdMob.

3. Faixa Etária

Este aplicativo é destinado a usuários com 18 anos ou mais. Não coletamos intencionalmente informações de crianças ou adolescentes.

4. Permissões

O aplicativo solicita apenas as permissões estritamente necessárias para a leitura do banco de dados bíblico e funcionamento da interface offline.
